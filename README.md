# 🧮 مساعد المحاسبة — Accounting Mentor

تطبيق تعليمي تفاعلي بالكامل بالعربية يعلّمك **كيف تفكّر كمحاسب محترف** — لا حفظاً بل فهماً — يعمل بالكامل بدون إنترنت (Offline First) وقابل للتثبيت كتطبيق Android عبر Capacitor.

---

## ✨ المميزات

- 🔍 **تحليل ذكي بالعربية**: أدخل معاملة بلغة طبيعية عربية، وسيقودك التطبيق عبر **8 خطوات تفكير محاسبي احترافي**.
- 👨‍🏫 **وضع التدريب**: سيناريوهات موجّهة مع تلميحات وشرح تدريجي.
- 📝 **وضع الاختبار**: أسئلة عشوائية مع تقييم فوري وحفظ التقدم.
- 📚 **قاعدة معرفة**: دليل حسابات + قواعد ذهبية + أفعال محاسبية.
- 📜 **سجل كامل**: كل تحليلاتك محفوظة محلياً وقابلة للبحث.
- ⚙️ **إعدادات كاملة**: ثيم فاتح/داكن/نظام، حجم خط، سرعة حركة.
- 🎨 **تصميم Material 3** مع دعم RTL وخط Cairo.
- 📱 **يعمل بدون إنترنت**: كل شيء محلي عبر IndexedDB.

## 🚀 التشغيل السريع (تطوير)

**متطلبات**: Node.js 18+ و npm.

```powershell
npm install
npm run dev
```

ثم افتح [http://localhost:3000](http://localhost:3000).

---

## 📦 بناء تطبيق أندرويد (APK) — دليل كامل لويندوز 10

راجع [`APK-BUILD-GUIDE.md`](./APK-BUILD-GUIDE.md) للحصول على تعليمات خطوة بخطوة.

باختصار:

```powershell
# 1. تثبيت الاعتماديات
npm install

# 2. بناء المشروع كملفات ثابتة
npm run build

# 3. إضافة منصة أندرويد (لأول مرة فقط)
npx cap add android

# 4. مزامنة الملفات مع مشروع أندرويد
npx cap sync android

# 5. بناء APK
cd android
.\gradlew.bat assembleDebug
```

سيكون ملف APK جاهزاً في:
```
android\app\build\outputs\apk\debug\app-debug.apk
```

## 🧠 كيف يعمل محرك التحليل؟

1. **Parser**: يُطبّع النص العربي (إزالة تشكيل، توحيد ألف/همزة/ياء/تاء مربوطة)، ويستخرج المبلغ وطريقة الدفع.
2. **Classifier**: يبحث في قاعدة المعرفة عن الحسابات والأفعال المطابقة عبر الكلمات المفتاحية والمرادفات.
3. **Journal Generator**: يستنتج اتجاه التغيير لكل حساب (زيادة/نقص) بناءً على الفعل، ثم يطبّق **القواعد الذهبية** لتحديد المدين والدائن، ويولّد قيداً متوازناً.
4. **Analyzer**: ينسّق كل ما سبق في **8 خطوات تعليمية** يمرّ بها المستخدم واحدة تلو الأخرى.

- تشغيلي (إيجار، كهرباء، نقل)

- تعليمي (رسوم دراسية)
- تمويلي (قروض، رأس مال)
- حقوق ملكية (مسحوبات)
